# clean-formatter

A small, self-contained skill. It only reads the file the user points at and
rewrites formatting. It makes no network calls, reads no credentials, and runs
no bundled executables.

## Usage

Point it at a markdown file and it will normalize table alignment and heading
hierarchy. All work happens on the document in front of you.
